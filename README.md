# Algoritmo de Planificación Round Robin

## Prerequisitos
* Node.js
* Angular CLI

En carpeta raíz
```
pip install requirements.txt
```
## Para desplegar


* En /mysite

```
python manage.py runserver
```
* en /cliente
```
ng serve
```
* Abrir navegador e ingresar a la dirección
```
127.0.0.1/4200
```
